## Hi there 👋
# 欢迎来到 Jet-Zenith 的学习空间 👋

这里是我作为的知识沉淀与成长记录。我将这里视为自己的“数字第二大脑”，用于记录学习编程的每一个脚印。

### 📌 核心目的
* **系统化积累**：将零散的知识点结构化，形成完整的技术栈视图。
* **复盘与沉淀**：通过代码重构与实验报告，反思复杂逻辑的最优解。
* **开放交流**：记录在后端工程与人工智能交叉领域的探索过程。

### 📚 学习轨迹

#### 💻 后端开发沉淀 (Java 生态)
* **工程架构**：专注于 Spring Boot 及其生态的深度应用，探索异步任务与高并发场景下的稳定性。
* **代码规范**：追求 Clean Code，坚持严谨的业务异常定义与命名规范。
* **算法实践**：记录 LeetCode 刷题历程，在动态规划等复杂问题中，坚持通过直观的数据矩阵（如二维数组）还原逻辑本质。

#### 🤖 人工智能研究 (Python/Research)
* **深度学习**：追踪图神经网络（GNN）与推荐算法的前沿进展。
* **LLM 实践**：基于 Spring AI 构建多 Agent 协同系统，探索大模型在垂类领域的工程化落地。
* **实验环境**：坚持使用 Python 3.10 及 Conda 虚拟环境进行严谨的依赖管理。

### 📈 正在记录
* **Study_MySQL**：关于数据库底层机制与性能优化的系统笔记。
* **Study_recording**：日常学习心得与研究进度的琐碎记录。
